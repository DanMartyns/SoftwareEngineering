1. RUN THE PORTAINER

    -> docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock portainer/portainer

2. CREATE DOCKER WITH Jenkins

    -> docker run --name jenkins -p 8080:8080 jenkins/jenkins:lts

3. CREATE DOCKER WITH MySQL

    creates a MySQL running in a container name SoftwareEngineering. 
    The file system will be mapped on local host c:\\tmp\\mysql
            It will create "SoftwareEngineering" database on startup.
            The root password will be "es"
            It will create an user "User" with password "es"
            It will export the port 3306 of the container ( default port on mysql is 3306 ) 


    -> docker run --name SoftwareEngineering -e MYSQL_ROOT_PASSWORD=es -e MYSQL_DATABASE=database -e MYSQL_USER=User -e MYSQL_PASSWORD=es -p 3306:3306 -d mysql:latest


3.1 ACCESS DATABASE

    To access the database write in the terminal the following commands :  

    -> docker exec -it SoftwareEngineering bash
    -> mysql -u User -p
    
    In the step above, the database will ask you for a password. The value of the password is "es"
    After entering the database, write the following commands if you want to see the content of the table, for example :
    
    -> use database;
    -> show tables;
    -> SELECT * FROM ecg;
